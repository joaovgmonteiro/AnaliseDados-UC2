Variáveis Qualitativas: atributos que não podem ser medidos numericamente. (Classificação - categorizar)
Ex.:cor_olhos = "castanho"
    estado_civil = "solteiro"
    tipo_veiculo = "carro"

Variáveis Quantitativas: Expressam números.
Ex.:idade = 28
    altura_cm = 175
    renda_mensal = 3500.0

Conceitos Estatísticos
População: Conjunto total de individuos com caracteristicas em comum. Ex: Todos os habitantes de uma cidade 
Censo: levantamento completo de dados de todos os elementos da população. Ex: Pesquisa do IBGE que coleta dados de todos os 
brasileiros.
Amostra: Subconjunto de uma população, selecionado para análise. Ex: 1000 pessoas escolhidas para responder uma pesquisa 
de opinião.
Amostragem: Processo/técnica para selecionar a amostra de uma população.
